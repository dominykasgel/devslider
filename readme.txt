A very simple responsive slider for WordPress. Built for developers.

== Description ==

DevSlider is a simple slider plugin that allows you to create slides, customize the slider template and extend the slider by adding new options. It has many hooks and filters.

**Awesome Features**

* Uses The Responsive jQuery Content Slider - [bxSlider](http://bxslider.com/)
* Has many options - [bxSlider examples]http://bxslider.com/examples
* Easy to modify and override the slider template
* Easy to extend the slider by adding new options
* Built for developers
* Has many filters and hooks

== Installation ==
1. Upload `devslider` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
